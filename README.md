# 看拼音写字 · 练习页生成器

一个**单文件 HTML 工具**，输入中文词汇即可生成可打印的「看拼音写字」练习页。无需安装、无需联网，用浏览器打开即可使用。

## 功能特点

- **词汇输入**：支持空格、英文逗号、中文逗号分隔词语
- **章节划分**：用单独一行的 `-----`（至少 5 个连字符）分隔内容，自动生成「第1章」「第2章」……标题
- **拼音转换**：内嵌 [pinyin-pro](https://github.com/zh-lx/pinyin-pro)，离线可用，准确率高
- **书写格样式**：田字格、米字格、方框、横线
- **拼音声调**：带声调（nǐ hǎo）、数字声调（ni3 hao3）、无声调（ni hao）
- **答案页**：可附加淡色汉字参考答案，打印时单独分页
- **打印优化**：A4 纵向排版，含姓名 / 日期栏

## 快速开始

1. 下载或克隆本仓库
2. 用浏览器打开 [`kan-pinyin-xiezi.html`](./kan-pinyin-xiezi.html)
3. 输入词汇，点击「生成练习页」
4. 点击「打印」输出练习纸

> 支持 Chrome、Edge、Safari、Firefox 等现代浏览器。

## 词汇输入

### 基本格式

词语之间用**空格**或**逗号**分隔：

```
你好 学校 中国 美丽 快乐
```

也支持中文标点，会自动拆分：

```
你好，学校、中国；美丽
```

### 章节分隔

单独一行输入至少 5 个连字符，可划分多个章节：

```
你好 学校
-----
中国 美丽 快乐
-----
春天 学习 朋友
```

生成效果：

- **第1章**：你好、学校
- **第2章**：中国、美丽、快乐
- **第3章**：春天、学习、朋友

未使用 `-----` 分隔时，不会显示章节标题。

## 可配置选项

| 选项 | 说明 |
|------|------|
| 练习标题 | 页面顶部标题，默认「看拼音写字」 |
| 格子大小 | 小 / 中 / 大 / 特大 |
| 书写格样式 | 田字格、米字格、方框、横线 |
| 拼音声调 | 带声调、数字声调、无声调 |
| 显示姓名 / 日期栏 | 打印时显示填写栏 |
| 词语分组显示 | 同一词语的字保持为一组 |
| 附加答案页 | 在练习页后附加淡色汉字参考答案 |

修改选项后，若已生成过练习页，预览会**自动更新**。

## 快捷键

- `Ctrl + Enter`（Mac：`Cmd + Enter`）：快速生成练习页

## 打印建议

1. 纸张：A4 纵向
2. 边距：使用浏览器默认即可（已针对约 14mm 边距优化）
3. 开启答案页时，参考答案会**单独打印在下一页**
4. 建议勾选「背景图形」，以确保淡色答案汉字正常输出
